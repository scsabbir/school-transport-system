# 🚍 School Transport Management System

A modern school transport management system.

## Features

- Admin Dashboard
- Parent App
- Driver App
- Live GPS Tracking
- Student Check-in / Check-out
- Route Management
- Push Notifications

## Tech Stack

- Next.js
- Flutter
- Node.js
- Express.js
- PostgreSQL
- Socket.IO
- Firebase Cloud Messaging
